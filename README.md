# Chat Client на пайтон
Простий графічний чат-клієнт з підтримкою:

- текстових повідомлень
- надсилання зображень
- зміни ніку
- анімованого бокового меню
Інтерфейс створений за допомогою CustomTkinter
# Можливості

 Відправка текстових повідомлень  
 Відправка зображень  
 Відображення отриманих картинок  
 Зміна імені користувача  
 Scrollable чат  
 Адаптивний UI  
 Підключення до TCP сервера  .

 # 🛠Використані бібліотеки

- threading
- customtkinter
