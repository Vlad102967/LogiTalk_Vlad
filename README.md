# LogiTalk_Vlad
Код был написан на языке програмировани Python.В него входят библиотеки CustomTkinter,socket и reading.Его можно использовать как локальный мессенджер для простого общения.

# Запуск программы
LogiTalk 

# История создания
Код был создан на курсе програмирования Logika

# Функционал Клиента
Функционал включает в себя функции такие как:подключение к серверу,смена и сохранение никнейма,отправка сообщений и фото
